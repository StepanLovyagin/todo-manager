# Практическая работа №13 — React To-Do App

Приложение создано на базе Vite + React. 

### Что сделано:
- Основной функционал: добавление, удаление, отметка выполнения задач.
- Хранение данных: используется `localStorage`, чтобы задачи не пропадали после обновления страницы.
- Фильтрация: можно смотреть все задачи, только активные или только выполненные.

### Дополнительные функции:
1. **Редактирование:** Если дважды кликнуть по тексту задачи, откроется поле для изменения текста.
2. **Тёмная тема:** Добавлена кнопка переключения ночного режима для удобства глаз.
